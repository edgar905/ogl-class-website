AI for Professionals:

A four-week live cohort for professionals who want to go from "I've heard about AI" to "I shipped something with it." Each session is 90 minutes. Each week you build something real. Classes run the first Wednesday of every month.

The Four Stops

Your mission map, week by week.

Each week builds on the last. You're not just learning — you're navigating a route, and we're your crew.

Week 01 — Launch Pad

🚀 AI Fundamentals

Before you can fly, you need to understand the machine. Week 1 covers how large language models actually work — not the hype, the mechanics. You'll leave with a clear mental model of what Claude (and AI in general) can and can't do, and a set of prompting fundamentals that will accelerate everything else.

How LLMs work — tokens, context, and why outputs vary
Claude's strengths and real-world use cases for professionals
Prompt structure: roles, context, output formats
Live exercise: build a prompt that solves a real problem at your job
⏱ 90-minute live session + workbook + replay
Week 02 — First Orbit

⚡ Claude Co-Work & Code

You're in orbit. This week you pair with Claude Code for the first time — opening a terminal, working with files, and writing your first automation. We cover file structure basics so nothing feels mysterious. You'll do a live build alongside the instructor and leave with working code.

Setting up Claude Code — environment walkthrough
File structure basics: folders, naming conventions, version control thinking
Live co-work session: write a real task with Claude step by step
Q&A: what to do when Claude gets it wrong
⏱ 90-minute live session + code workbook + replay
Week 03 — Deep Space

🛰️ GitHub, Skills & Projects

Deep space: the tools get more powerful and the builds get more serious. Week 3 is where your workflow becomes something portable. We cover how to pull skills, create projects, and understand GitHub well enough to never lose your work again.

GitHub basics: repositories, commits, branching (no prior experience needed)
Pulling and using Claude Code skills from a repository
Creating a project: structure, naming, documentation
How-to: sharing your work and getting feedback before Week 4
⏱ 90-minute live session + project template + replay
Week 04 — The Landing

🌍 Build, Test & Troubleshoot

The landing. Week 4 is where everything comes together. You build a complete project from start to finish, run it, test it, break it on purpose, and fix it. We cover the debugging mindset and the habits that separate reliable automation from a one-time experiment.

Complete project build: choose your use case, build it live
Testing methodology: how to verify AI output is doing what you expect
Troubleshooting playbook: the five most common Claude Code failures and how to fix them
Where to go next — resources, communities, and continued learning
⏱ 90-minute live capstone session + replay

FAQ

Common questions.

Do I need coding experience to take this class? ↓

No. The class is designed for professionals, not developers. If you can use a browser and follow written instructions, you have enough technical foundation. Week 2 introduces command-line basics from scratch — we don't assume you've touched a terminal before.

What happens if I miss a live session? ↓

Every session is recorded and added to the replay library within 24 hours. You get lifetime access to all four recordings. Missing a live session doesn't block your progress — you can catch up before the next week and still submit questions via email.

When exactly are the live sessions? ↓

Each cohort begins on the first Wednesday of the month at 12:00 PM EST / 9:00 AM PST. The following three Wednesdays are weeks 2, 3, and 4. The exact time is confirmed when you enroll — we send a calendar invitation with all four dates.

What's the difference between the Professional and Executive tiers? ↓

Professional adds a single 30-minute one-on-one session where you can work through a specific workflow or question with the instructor. Executive adds two 60-minute consulting sessions, a custom review of a workflow you're building for your job, and priority email support throughout the cohort.

What tools will I need to have installed? ↓

For weeks 1–2: just a browser. For weeks 3–4: we'll walk you through installing Claude Code and setting up a free GitHub account. All the tools are free. We send setup instructions the week before the cohort starts so you're ready on day one.

Can I purchase the Enterprise tier for a remote team? ↓

Yes. Enterprise workshops are conducted virtually via Zoom and can accommodate teams across multiple time zones. We'll work with you to find session times that work for your team. Email us to start the conversation about scheduling and pricing.
